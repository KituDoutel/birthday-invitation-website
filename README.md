This birthday invitation website was created with love for **Ryana Duta Lopes** to celebrate her birthday at **Suai Loro Beach, Timor-Leste**.

**I (KituDoutel)** developed this website with:

## 🎯 Implemented Features

<div align="center">

| ✅ Feature | 📝 Description | 🎨 Style |
|-----------|---------------|----------|
| **Responsive Design** | Adapts to all screens (mobile, tablet, desktop) | CSS Media Queries |
| **Countdown Timer** | Real-time countdown (days, hours, minutes, seconds) | JavaScript |
